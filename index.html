This project was a Pair Coding project between Jeff Lucero and the Cursor AI platform. Frontend: HTML5, CSS3, JavaScript (ES6+) Tailwind CSS (via CDN) for utility-first styling Font Awesome 6.0 for icons Inter (Google Fonts) for typography Custom CSS (styles.css) for additional effects and theming Testing & Automation: Cypress (JavaScript based end-to-end testing framework) Selenium WebDriver (Python-based broswer automation) Pytest (Python based test management and reporting framework)
