Jeff Lucero and Cursor AI building a better future... together! Build Together.

This project was a Pair Coding project between Jeff Lucero and the Cursor AI platform. Frontend: HTML5, CSS3, JavaScript (ES6+) Tailwind CSS (via CDN) for utility-first styling Font Awesome 6.0 for icons Inter (Google Fonts) for typography Custom CSS (styles.css) for additional effects and theming Testing & Automation: Cypress (JavaScript based end-to-end testing framework) Selenium WebDriver (Python-based broswer automation) Pytest (Python based test management and reporting framework)

Why choose me for your AI Product Builder role?

I built this site in a few hours over a weekend using Cursor AI! But it showcases my ability to connect an automated QA framework to cover the functionality of the same application! Using Selenium and Cypress, you can quickly build a very robust and comprehensive test suite for your platform.

Pros: Quick Debugging of the Application

As I'm building the code and executing commands in the terminal, Cursor and I can knock down and debug everything from: terminal errors, github auth issues, npm install issues, CSS look and feel issues, npm dependencies issues, and more! As a AI product builder, I can focus on building and testing the functionality of the application, and Cursor can focus on the details of the code.

Con: Cypress and Browser Rendering Issues

Cursor can't interpret the front end test results, a human being still needs to do that, but the AI can write the tests very quickly, shifting my QA focus to what actually matters: why is the functionality test failing?
